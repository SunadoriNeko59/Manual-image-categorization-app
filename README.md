# 🌟 2値BMP マルチクラス分類ツール

このツールは、複数枚の画像（主に領域抽出された二値画像など）を読み込み、様々なロジック（K-Means法、重なり領域、白ピクセル率、マスク判定など）を用いて自動的に複数クラスに分類できるGUIアプリケーションです。

分類結果は視覚的に分かりやすい「ツリーダイアグラム（組織図形式）」として表示され、画面上での手動調整（ドラッグ＆ドロップ）や、最終的な振り分け結果のExcelフォーマットでの出力が可能です。

## ✨ 主な機能

- **🌲 ツリー形式の直感的なUI:** 分類結果を階層ツリー型で可視化。ツリーノードのマウス操作で直感的に手動での再分類調整が可能。
- **🧩 プラグインによる高い拡張性:** 分類アルゴリズムはすべて `plugins/` ディレクトリ内の別ファイルとして分離されており、独自のカスタム分類ロジックを簡単に追加できます。
- **🎨 リアルタイムプレビュー機能:** 選択した画像の重なりを画面上の赤い半透明オーバーレイで確認。マスクファイルを用いた分類時には黄色のオーバーレイで対象箇所を視覚的に表示します。
- **↩️ Undo/Redo対応:** 誤った操作を取り消したり、やり直したりすることで安心して作業を進められます。
- **📊 Excelエクスポート:** 最終的な分類ツリー構造全体や、各画像の所属クラス一覧を Excel (`.xlsx`) ファイルとしてエクスポートします。

## 📦 組み込み済みの分類機能 (プラグイン群)

アプリケーションには標準で以下の多様な分類ロジックが組み込まれています。

- **K-Means (見た目で分ける):** ピクセル情報を元に、指定したクラス数に自動で画像を画像クラスタリング。
- **白の面積で分ける:** 画像全体の白ピクセル（255）の面積比率を計算し、指定したパーセンテージ範囲で画像を段階的に分割。
- **白率判定 (0%とそれ以外):** 真っ黒の画像（白ピクセルが0%）と、それ以外の画像（少しでも白が含まれる画像）とに2分割。
- **重なり領域 (位置で分ける):** 指定した座標や領域に対する重なりの大きさで画像をクラスタリング。
- **外周判定 (枠に白ありなし):** 画像の外枠部分に白ピクセルが含まれているかどうかで2つに判定・分割。
- **マスク画像判定:** ユーザーが指定した基準となる「マスク画像」と照らし合わせ、その領域内に含まれる白の割合で画像を分類。

## 🚀 インストールと起動方法

1. リポジトリをダウンロードまたはクローンします。
2. 必要なPythonパッケージをインストールします。

```bash
pip install -r requirements.txt
```

3. アプリケーションをメインスクリプトから起動します。

```bash
python main.py
```

## 📂 アーキテクチャ (ディレクトリ構成)

本ツールのディレクトリ構成は、保守性と拡張性を高めるため、UI・ビジネスロジック・プラグインがきれいに分離されています。

- `main.py`: アプリケーションの起動用エントリポイント
- `src/core/`: ビジネスロジック群
  - `state.py`: （AppState）画像リストや分類ツリー状態などのアプリケーションデータ・状態管理
  - `classifier.py`: （ImageClassifier）画像オーバーレイ合成処理・分類サポート処理
  - `plugin_manager.py`: `plugins/` フォルダ内の外部プラグインの動的読み込みと実行管理
- `src/ui/`: ユーザーインターフェース (Tkinter)
  - `main_window.py`: メインウィンドウの構築およびUIイベント制御
  - `tree_renderer.py`: ツリーダイアグラム専用の描画制御クラス
  - `dialogs.py`: 自動分類など各種設定機能用ポップアップダイアログ
  - `constants.py`: ツールのUI定数（カラーコード、表示サイズ等）群
- `src/utils/`: 共通ユーティリティ
  - `logger.py`: カスタムの標準ロガー
  - `image_utils.py`: 画像処理・変換に関する独立したヘルパー関数
- `plugins/`: 拡張分類アルゴリズム（プラグインスクリプト）の配置先

## 🛠️ プラグインの追加と作成方法

独自の分類ロジックを追加したい場合は、`plugins/` フォルダ内に新しい Python スクリプトのファイルを作成するだけです。
アプリケーションのUIの「自動分類」ダイアログの選択肢に、作成した分類ロジックが動的に追加されます。

### プラグイン作成用のテンプレート

```python
from PIL import Image
from typing import List, Any

# 関数名は必ず `classify` と定義してください
def classify(images: List[Image.Image], state: Any, k: int) -> List[int]:
    """
    対象画像を分類するロジックを実装します。

    Args:
        images: 分類対象となるPIL画像のリスト
        state: AppStateインスタンス（現在のアプリの各種データ・設定）
        k: ユーザーがUIで指定した「分割したいクラス（グループ）数」
    Returns:
        各画像が所属するクラスインデックス（0 ~ k-1）のリスト
    """
    labels = []
    
    for img in images:
        # ここに実装したい独自判定・分類ロジックを記述します
        labels.append(0)  # 例: 今回はすべてをクラス0に判定
        
    return labels
```
